Website
Mylinktree

MyLinkTree adalah proyek website pribadi yang dibuat menggunakan HTML dan CSS. Tujuannya untuk mengumpulkan semua tautan media sosial dan link penting dalam satu halaman, sehingga pengguna lain dapat menemukanmu dengan mudah hanya melalui satu link utama.

Tentang Proyek

Website ini merupakan proyek MyLinkTree yang dibuat oleh Sugiati sebagai wadah untuk menampilkan berbagai tautan penting dan media sosial dalam satu halaman yang ringkas dan menarik. Tujuannya untuk memudahkan pengunjung menemukan semua link hanya melalui satu halaman utama.

Proses dan Teknologi

  • Dibangun menggunakan HTML dan CSS murni tanpa framework berat.
  • Struktur grid sederhana menggunakan Bootstrap Grid System.
  • Desain minimalis dengan warna lembut agar nyaman dilihat.

Tujuan

  • Menampilkan identitas digital dan tautan media sosial dalam satu halaman.
  • Melatih kemampuan dasar dalam desain web statis.
  • Membuat tampilan yang responsif di berbagai perangkat.
